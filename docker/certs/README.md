# SSL Certificates

This directory holds locally-trusted TLS certificates generated by [mkcert](https://github.com/FiloSottile/mkcert).

The certificate files (`*.pem`) are **gitignored** — each developer generates their own.

## Setup

```bash
# 1. Install mkcert (once per machine)
brew install mkcert
mkcert -install          # installs the local CA into your system trust store

# 2. Generate certs (from the project root)
just ssl                 # generates seed.pem and seed-key.pem in this directory

# 3. Add the domain to /etc/hosts
echo "127.0.0.1  seed.local" | sudo tee -a /etc/hosts

# 4. Restart nginx
just restart nginx
```

After this, `https://seed.local` will work in your browser with a green padlock.

## Files

| File | Description |
|---|---|
| `seed.pem` | Certificate (gitignored) |
| `seed-key.pem` | Private key (gitignored) |

## Without SSL

The stack works over plain HTTP too. In `nginx/app.conf`, comment out the HTTPS
server block and the HTTP redirect block, then add a plain `listen 80` server block.
